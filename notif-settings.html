School Operator

Notification Settings

Configure SMS and email providers for parent notifications

sms

SMS Provider

Used for sending fee reminders and alerts to guardian phone numbers

This is what appears as the sender on the parent's phone. No spaces allowed.

Getting your Termii API key:

  1. Go to app.termii.com and create an account
  2. Verify your business and register your Sender ID
  3. Copy your API key from Settings → API Keys
  4. Top up your SMS balance (₦1–₦3 per SMS depending on volume)
email

Email Provider

Used when channel is set to Email or SMS + Email

Getting your Resend API key:

  1. Go to resend.com and create a free account
  2. Add and verify your school's domain (e.g. greenwoodacademy.edu.ng)
  3. Create an API key under Settings → API Keys
  4. Free tier: 3,000 emails/month — more than enough for most schools
bolt

Edge Function Deployment

The send-notifications Edge Function must be deployed to your Supabase project

Run these commands from your project root to deploy the notification sender:

# Install Supabase CLI if not already

npm install -g supabase

# Link to your project

supabase link --project-ref your-project-ref

# Set secrets (your API keys)

supabase secrets set TERMII_API_KEY=your-termii-key

supabase secrets set RESEND_API_KEY=your-resend-key

# Deploy the edge function

supabase functions deploy send-notifications

The edge function code is in: edge-functions/send-notifications/index.ts
